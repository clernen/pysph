Nearest Neighbor Particle Searching¶

The interaction of particles in SPH is limited to neighboring particles that are within a certain cutoff radius from the target particle as shown in the figure below.

Interaction radius for a target particle

The all-pair search is very inefficient O(n^2) and can be reduced to O(nlogn) with the use of an efficient indexing scheme.

The cell structure¶

PySPH imposes a cell structure over the particle distribution as shown in the figure. The cell size is fixed at the start of each iteration and cells are only created where particles exist.

Using this cell structure and the cell to which the target particle belongs, we need only consider neighboring cells to search for neighbors.
