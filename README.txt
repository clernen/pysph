Welcome to PySPH
================

PySPH is an open source, parallel, framework for Smoothed Particle
Hydrodynamics (SPH) simulations. It is implemented in Python and the
performance critical parts are implemented in Cython. The framework provides
for load balanced, parallel execution of solvers and also supports running on
gpus supporting opencl. It is designed to be easy to use and extend.
